# 蒸気鉄道デュエル（Steam Rail Duel）

CPU（AI）と1対1で対戦する、鉄道経営のターン制ストラテジーゲームです。
線路を敷いて荷物（キューブ）を都市へ運び、収入と得点（VP）を競います。全8ターン。

▶ **プレイはこちら**: https://supershitakeman.github.io/steam-rail-game/

ブラウザだけで動きます（インストール不要）。

## 遊び方

1. ページを開くと、最初に**先手/後手**などの設定画面が出ます。
2. 各ターンは「アクション選択 → 建設（線路敷設）→ 輸送（荷物配送）」の流れで進みます。
3. **線路**を敷いて都市どうしをつなぎ、色の合う荷物を運ぶと**収入**が上がります。
4. 自分の線路を使って運ぶほど有利（相手の線路を使うと相手に収入が入ります）。
5. **機関車**を強化すると、より長い経路（多リンク）で運べるようになります。
6. 全8ターン終了時、**得点（VP）が高い方の勝ち**です。

操作はすべて画面上のクリックで行えます。

## 記録されるデータについて

CPU（AI）の改善のため、**同意いただいた対局のみ**、以下のデータを記録します。

- 対局の日時
- 勝敗・得点（VP）・ターン数・どちらの色を担当したか
- 対局の手のログ（盤面の操作記録）

**氏名・メールアドレス等、個人を特定する情報は一切収集しません。**

- 初回アクセス時に**同意ダイアログ**が表示され、「同意して記録する」を選んだ場合のみ記録されます。
- 「記録しない」を選べば、一切送信されません。
- 設定を変えたい場合は、ブラウザのサイトデータを消去してください。
